Default styles

Generates the default stylesheet for dots and navigation.

TinSlide(document.getElementById('slider'), {
  generate: {
    styles: {
      on: true
    }
  }
}

Wrapping element

Make sure you wrap the slider with a <div> or something. TinSlide will add position: relative to it, making the arrows center nicely.
