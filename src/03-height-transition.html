Height transition

TinSlide does not animate the slider height, but you can instruct TinSlide to explicitly set the slider height, and then use your own css transition.

#slider {
  transition: height 200ms cubic-bezier(.48,.01,.21,1);
} TinSlide(document.getElementById('slider'), {
  useUpdateContainerHeight: true,
  verticallyCenter: true,
  generate: {
    dots: {
      on: false
    },
    nav: {
      on: false
    }
  }
});
